# Job Application Compass Skills v0.1.0-skills.1

This is the first public skills-only release for Job Application Compass.

## Included

- `resume-crafter`
- `cover-letter-crafter`
- `portfolio-crafter`
- `communication-story-crafter`
- Individual `.skill` and `.zip` downloads for each skill.
- `job-application-compass-skills-plugin.zip`, a skills-only bundle for hosts
  that support plugin-style imports.

## Not Included

- The private Job Application Compass MCP.
- The local packet engine.
- Profiles, source registers, job-hunt state, or sample application packets.
- Any auto-apply, CRM, tracker, scraping, or submission workflow.

## Boundary

The skills help a host AI craft career materials from supplied facts. They
should preserve missing information as visible gaps and should not invent
experience, metrics, credentials, dates, company research, or project outcomes.
