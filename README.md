# Job Application Compass Skills Public Page

This repository publishes the public, skills-only layer of Job Application
Compass.

It does not publish the private Job Application Compass MCP, packet engine,
profiles, source register, sample application packets, local job-hunt state, or
any private evidence files.

## Release

- Version: `v0.1.0-skills.1`
- Live site: `https://jonathankhobson.github.io/job-application-compass/`
- Release assets: individual `.skill` and `.zip` packages for four public
  skills, plus `job-application-compass-skills-plugin.zip`.

## Public Skills

- `resume-crafter`
- `cover-letter-crafter`
- `portfolio-crafter`
- `communication-story-crafter`

## Boundary

These skills are prompt guidance. They help a host AI work from supplied facts.
They do not run local tools, submit applications, scrape jobs, maintain a
profile, or expose the private packet engine.
