What is Job Application Compass Skills?

Job Application Compass Skills is the public, prompt-only layer of a private job-application system. This release includes four generic skills that help an AI build, edit, target, and audit career materials from facts the user supplies.

The private local MCP, packet engine, personal profile, source register, sample applications, and job-hunt state are not included in this public release.

About These Skills

These skills are generic public extractions from Job Application Compass. They preserve the reusable writing methods while leaving out private personal data and the local packet-generation MCP.

They are designed for AI hosts that can read Markdown skills. The skills guide drafting, editing, routing, and audits, but they do not fetch private files, maintain a profile, track applications, or submit anything for the user.

Evidence First

The skills treat supplied facts as the boundary. If a credential, metric, company signal, project outcome, or ownership claim is missing, the output should ask for it or mark it visibly.

Human And ATS Readers

Resume guidance balances parser-friendly structure with recruiter clarity. Cover letters, portfolios, and messages prioritize proof, voice, and reader fit without slipping into generic polish.

Private MCP Not Included

The private Job Application Compass MCP remains local-only. It includes packet builders, source authority, profiles, and job-specific workflows that are not public download material.

Install Guide

This release ships skills only. There is no Claude Desktop Extension, MCPB, local server, or source install path for the private packet engine.

Skill File

  1. Download the .skill file for the lane you want.
  2. Add or import it through your host app's skill flow.
  3. Ask the host AI to use that named skill for your resume, cover letter, portfolio, or communication task.

Skill Zip

  1. Download the matching .zip file.
  2. Inspect the Markdown if you want to review the instructions first.
  3. Import the zip if your host expects a compressed skill folder.

Skills-Only Plugin

  1. Download job-application-compass-skills-plugin.zip.
  2. Use it only in hosts that support plugin-style skill bundles.
  3. Confirm the package exposes skills only. It should not ask you to run a local MCP server.

Example

Public-safe examples are intentionally blank for this MVP. The private Job Application Compass sample packets contain personal career material and are not used as public demos.

Future examples should use fictional or fully consented material and should prove that the skills preserve missing facts instead of smoothing them into unsupported claims.

What do these skills actually do?

They give an AI assistant structured guidance for resume writing, cover letters, portfolio case studies, and career communication. They help the AI ask for missing facts, use stronger patterns, and avoid invented claims.

Why is there no MCP download?

The MCP is a private local packet engine that can touch personal profiles, source registers, and application files. This public release shares only the scrubbed generic skills.

What is the resume skill for?

Use Resume Crafter to build, target, audit, and improve resumes, including bullet rewriting, ATS cleanup, role positioning, metrics, and truth checks.

What is the cover letter skill for?

Use Cover Letter Crafter for full letters, hooks, proof paragraphs, company-fit review, voice cleanup, and unsupported company or metric claims.

What is the portfolio skill for?

Use Portfolio Crafter to turn project notes and artifacts into case studies, case cards, captions, leave-behind outlines, and confidentiality-aware audits.

What is the communication skill for?

Use Communication Story Crafter for ethical storytelling, hooks, audience adaptation, job-search messages, stakeholder updates, and voice or tone revision.

Will these apply to jobs for me?

No. They do not auto-apply, submit forms, scrape postings, send messages, or track applications. They help craft text from supplied facts.

Can they invent stronger metrics?

No. They can help recover verified metrics, transparently phrase estimates when the calculation path is supplied, or use honest non-numeric impact language.

Package Contents

The individual skill files contain one public skill each. The plugin bundle contains .claude-plugin/plugin.json, README.md, and skills/<skill-name>/....

It must not contain mcp/, .mcp.json, .mcpb, server.py, profiles, source registers, sample applications, or private packet artifacts.
